Lista de exercicios e aulas

Esta página foi criada para armazenar e revisar os conteúdos das aulas de Front-End Design Engineering ministradas por Jonas Kimio Isiki.

Na seção de aulas, você encontrará códigos copiados e anotações das aulas, tanto online quanto presenciais. Os exercícios incluem códigos que eu, Jonas, desenvolvi ao realizar as atividades entregues nas aulas, abrangendo exercícios e tarefas avaliativas.

Infelizmente, nem todas as atividades e páginas estão liberadas, pois não as realizei ainda ou eu esqueci de salvar na máquina. (A atividade 2 não é possivel acessar pois essa página que você esta lendo agora é a atividade 2) todas as atividades, aulas e provas que não consegui colocar no site eles estarão visiveis mas não interagiveis

Futuramente pretendo poder substituir as fotos do código para um meio de vocês poderem copiar o código com um apertar de um botão. mas por agora o que eu recomendo é:

Utilizando o F12 pelo chrome ou Botão Direito do Mouse + Inspecionar você poderá ver o código do exercicio caso a foto do código não esteja completo ou deseja ler o código mais detalhado (que no caso nem sempre está completo e a foto pode estar um pouco borrada)

No código você verá esse simbulo "< ! -- (texto) -- >" cujo são os comentários que utilizamos para explicar cada comando no código das aulas apenas utilize eles se queira colocar um comentário no código pois qualquer coisa que você coloque dentro do código não irá rodar

Este site será constantemente atualizado com várias funções que estou aprendendo nas aulas e que irei pesquisar na internet. Aproveito para pedir desculpas pela TOTAL DESORGANIZAÇÃO dos arquivos no GitHub; ainda não entendo completamente como o sistema de commits funciona.
